# 🚀 Projeto Competitiva OBI

Bem-vindo(a) ao repositório que ajuda iniciantes a se prepararem para as Olimpíadas Brasileiras de Informática (OBI) com foco em desenvolvimento de raciocínio lógico, algoritmo e resolução de problemas.

> 🎯 Missão: juntos vamos transformar curiosidade em prática, permitindo que estudante de qualquer nível consiga aprender, praticar e evoluir de forma estruturada.

---

## 📚 O que o repositório oferece

- 🧩 Exercícios comentados e exemplos passo a passo
- 🖼️ Imagens e diagramas ilustrativos (fluxogramas, árvores, grafos)
- 🎥 Vídeos de aula e resolução de desafios com explicações detalhadas
- 📄 PDFs de teoria, listas de exercícios e gabaritos
- 📊 Apresentações (slides) para revisões e aulas em grupo

---

## ✨ Estrutura sugerida

- `imagens/` - figuras, diagramas, exemplos visuais
- `videos/` - gravações de aulas, screencasts de resolução de problema
- `pdfs/` - apostilas, cadernos de exercícios e resumos
- `apresentacoes/` - slides para ensino e revisão
- `codigos/` - soluções em C/C++/Python/Java para problemas de treino

---

## 🧠 Como usar

1. Comece pelos conteúdos básicos e teoria
2. Pratique com exercícios e compare com as soluções
3. Assista vídeos para entender a lógica e os padrões de resolução
4. Revise com slides e PDFs antes de simular provas

---

## 🏆 Resultado esperado

- Aprendizado passo a passo
- Crescimento de confiança para resolver problemas
- Preparação efetiva para competições de programação
