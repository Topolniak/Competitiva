O que são Primeiras Interações?

Após conhecer a plataforma beecrowd e preparar seu ambiente de programação, é hora de colocar a mão na massa. As primeiras interações envolvem resolver um problema simples de forma guiada, entendendo como o sistema recebe dados de entrada, como seu programa processa essas informações e como produzir a saída correta. Isso constrói a base para desafios mais complexos.

Recebendo Dados de Entrada

No beecrowd, os dados de entrada são fornecidos via entrada padrão (stdin). Em Python, usamos input() para ler linhas de texto. É importante converter para o tipo correto (int, float) se necessário. Lembre-se: a entrada pode ter múltiplas linhas, então leia quantas forem especificadas no problema.

Processamento das Informações

Após ler os dados, seu programa deve processá-los conforme a lógica do problema. Isso inclui cálculos, decisões condicionais e manipulação de dados. Mantenha o código claro e organizado para facilitar a depuração.

Produzindo a Saída Correta

A saída deve ser exatamente como especificado: formato, espaços e quebras de linha. Use print() em Python. Qualquer diferença resulta em erro. Teste com os exemplos fornecidos.

Problema Guiado: Soma Simples (1001)

Leia dois valores inteiros e apresente a soma deles.

Entrada

10
9
Saída

X = 19
Código em Python

# Lê o primeiro valor
a = int(input())

# Lê o segundo valor
b = int(input())

# Calcula a soma
soma = a + b

# Imprime o resultado no formato exato
print("X =", soma)

Teste este código no beecrowd enviando-o para o problema 1001. Observe como a entrada é lida, processada e a saída é produzida.
